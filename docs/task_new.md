# **target: Критерии перевода**

### **Критерии перевода студента на следующий модуль**

1. Посещен минимум 1 вебинар онлайн
2. Просмотрено 80% занятий и 80% контента (720 ед)
3. Решены все обязательные задачи (60)
4. Пройден текущий контроль
5. Пройдена рефлексия
6. Пройдена промежуточная аттестация (>=8 баллов)
